Алгоритм действий получения отчета:
1. Если ранее не загружали, то загружаем файл excel c информацией по всем картам в буфер
2. Создаем из буфера таблицу "Cards"
3. Загружаем файл excel полученный с информацией "Easy" по карте в буфер
4. Создаем из буфера таблицу "EasyCards"
5. Загружаем файл excel полученный с информацией от кредитной компании в буфер
6. Создаем из буфера таблицу "CreditCompany"
7. При условии, что все три выше указанные таблицы загружены и сформированы, приступаем к формированию отчета
8. Группируем таблицы в отчет
9. Полученную таблицу можем сохранить на Вашем компьютере в excel формате
Примечание: загрузка excel файлов возможна только в формате Excel 97-2003

Описание значения параметра flag в таблицу отчета:

flag=0 - Данные не определены, либо присутствует не соответствие данных, либо задвоеность данных
flag=1 - Проплачено, всё ОК !
flag=2 - К карте привязано несколько клиентов
flag=3 - Клиент отсутствует в БД "Cards" и определен по БД "EasyCards"
flag=4 - Клиенты в БД "EasyCards" не выполнившие проплату (в excel отчетe 2-й лист)
flag=5 - Имеется несколько одинаковых сделок в БД "CreditCard", к которым привязана 1 сделка в БД "EasyCards" (в excel отчетe 3-й лист)
flag=6 - Имеется несколько одинаковых сделок в БД "CreditCard", к которым привязано более 1 сделки в БД "EasyCards" (в excel отчетe 4-й лист)
flag=7 - Клиенты в БД "EasyCards", которых нет в CRM (в excel отчетe 5-й лист)
